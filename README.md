集成前端、服务器端及网络数据库设计开发旅行信息分享应用本项目采用前后端分离架构

前端使用React.js，后端使用Flask，数据库使用MySQL

基本内容和要求
1)编写客户端应用,能够在旅行前后以及过程中分享旅行信息。
2)用Python Flask开发旅行信息分享Web服务器。
3)用MySQL做旅行信息分享Web服务器的后台数据库。
4)旅游前的攻略收集和筛选、队伍组建。
5)旅游中的路线云备份和队伍管理。
6)旅游后资源分享和浏览、好友聊天互动等。


运行项目：
运行后端服务器：
travel_sharing_app_v0\backend\.venv\Scripts\python.exe travel_sharing_app_v0\backend\src\app.py
运行前端服务器
cd travel_sharing_app_v0\frontend
npm run dev
